Who Am I?
Vincent Fazio
C++ is a powerful general-purpose programming language. It can be used to develop operating systems, browsers, games, and so on.
Java is used as the server-side language for most back-end development projects, including those involving big data and Android development..
Python also is often used as a support language for software developers, for build control and management, testing, and in many other ways.
Haskell is used for business logic and data analysis, fast prototyping and enhancing existing software environments with correct code, performance and scalability.
Go is used for programming across large-scale network servers and big distributed systems
Dart is a client-optimized programming language for apps on multiple platforms. It is developed by Google and is used to build mobile, desktop, server, and web applications.
